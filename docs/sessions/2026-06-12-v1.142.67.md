# v1.142.67 (2026-06-12)

Three slices: METS 1.12 export /mets/{id}.xml (metsHdr + Dublin Core dmdSec + fileSec with checksums + structMap, for archival interchange) + oEmbed provider /oembed (let other sites embed a Heratio record card, JSON+XML, discovery link injected) + Browse by genre/form /genres (+ .json) completing the taxonomy-browse set with /themes and /places. All read-only, catch-all-safe.
