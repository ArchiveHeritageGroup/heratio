## Creating Snippets

Save interesting sections for later reference:

### Step 1: Mark Start Point

While playing, click **Mark Start** when you hear the beginning of the section you want.

### Step 2: Mark End Point

Click **Mark End** when the section finishes.

### Step 3: Save Snippet
```
┌─────────────────────────────────────────────────────────────┐
│  SAVE SNIPPET                                               │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  Start Time:    00:05:32                                    │
│  End Time:      00:06:15                                    │
│  Duration:      43 seconds                                  │
│                                                             │
│  Title:         [Interview about farming methods  ]         │
│                                                             │
│  Description:   [Describes traditional plowing    ]         │
│                 [techniques used in the 1950s     ]         │
│                                                             │
│              [Cancel]    [Save Snippet]                     │
└─────────────────────────────────────────────────────────────┘
```

## Troubleshooting
```
Problem                          Solution
───────────────────────────────────────────────────────────
Audio won't play              →  Check browser supports format
                                 Try refreshing page
                                 Check volume isn't muted
                                 
Waveform not showing          →  Wait for it to generate
                                 Large files take longer
                                 
Playback stutters             →  Check internet connection
                                 Let it buffer/load first
                                 
Snippet won't save            →  Check you have edit permission
                                 Ensure end is after start
```
