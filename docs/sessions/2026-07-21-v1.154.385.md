# v1.154.385 (2026-07-21)

fix(i18n): correct Afrikaans 'Validate' from 'Bevestig' to 'Valideer'. 'Bevestig' means confirm, a different action; af.json already renders validate as valideer everywhere else (Valideer CSV, Valideringswagelys, 'om te valideer'), so the bare Validate key was the lone semantic and consistency outlier. Found by diffing the 1430-unit human-authored Afrikaans XLIFF at rari29 against lang/af.json - 1421 of 1427 shared strings were identical, and this was one of only 6 differences
