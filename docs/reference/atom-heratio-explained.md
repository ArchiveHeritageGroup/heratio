AtoM Heratio refers to the legacy AtoM-AHG fork, which is not the same as the standalone Heratio platform. The term "Heratio" on its own always means the standalone Laravel platform, and it is not related to AtoM in any other way. The official documentation for Heratio is found at [https://heratio.theahg.co.za/help](https://heratio.theahg.co.za/help). AtoM Heratio has the base AtoM with Laravel plugins where Heratio is pure Laravel.
