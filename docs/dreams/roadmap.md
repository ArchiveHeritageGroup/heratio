# Heratio Roadmap

> North: **preserve it, link it, prove it, open it.** Every phase is a step toward a memory of human culture that never dies and belongs to everyone. Apex: [#1212](https://github.com/ArchiveHeritageGroup/heratio/issues/1212) - *that humanity never forgets*.

This roadmap ladders: **delivered platform -> intelligent -> operational & trusted -> connected -> north-stars -> apex.** Each item links its tracker issue.

## Phase 0 - Foundation (delivered)

A complete, standards-based GLAM platform, international by design (jurisdiction-neutral core, pluggable per-market compliance).

- Archival description (ISAD(G) + RiC), actors (ISAAR), repositories (ISDIAH), class-table-inheritance entity model
- Elasticsearch discovery, advanced search, GLAM browse
- Research portal (workspaces, projects, bibliographies, annotations, reproductions, bookings, ODRL rights)
- Spectrum 5.1 museum procedures; heritage accounting (GRAP 103 module for SA, pluggable for other markets)
- AI services via the AHG gateway: HTR, NER, condition scan, neural TTS docent, translation
- REST API (v1/v2), ingest wizard, scanner/capture pipeline, reports/dashboards, 221+ help articles
- **Exhibition digital twin** (roadmap [#1145](https://github.com/ArchiveHeritageGroup/heratio/issues/1145), delivered): builder, plan editor, first-person 3D walkthrough, authored audio tours, multi-user presence + live docent, live conservation-data overlay, simulation/forecast, twin analytics, in-twin AI recommendation, immersive XR/VR, photoreal scan shells + point clouds, open exports (IIIF / glTF scene / JSON-LD ExhibitionEvent), colour-managed renderer, windows/doors authoring

## Phase 1 - Intelligent (near term)

Make the platform *think with* the user. ~80% of the parts already exist; this phase mostly connects them.

- AI Cataloguer - scans in, draft ISAD/RiC records out ([#1196](https://github.com/ArchiveHeritageGroup/heratio/issues/1196))
- AI Curator-Docent - conversational, in-room guide ([#1185](https://github.com/ArchiveHeritageGroup/heratio/issues/1185))
- Researcher copilot - question in, annotated source set out ([#1198](https://github.com/ArchiveHeritageGroup/heratio/issues/1198))
- Generative exhibitions - AI auto-curates a themed show ([#1186](https://github.com/ArchiveHeritageGroup/heratio/issues/1186))
- Visitor heatmaps in the twin ([#1187](https://github.com/ArchiveHeritageGroup/heratio/issues/1187))

## Phase 2 - Operational & Trusted (mid term)

The twin becomes an operations console; every asset becomes verifiable.

- Live IoT sensor binding + conservation alerts ([#1188](https://github.com/ArchiveHeritageGroup/heratio/issues/1188))
- Conservation time-scrubber - provenance + forecast on a timeline ([#1189](https://github.com/ArchiveHeritageGroup/heratio/issues/1189))
- Collections-wide preservation triage ([#1200](https://github.com/ArchiveHeritageGroup/heratio/issues/1200))
- Provenance & authenticity layer - C2PA + AI-inference provenance ([#1201](https://github.com/ArchiveHeritageGroup/heratio/issues/1201), [#61](https://github.com/ArchiveHeritageGroup/heratio/issues/61))
- Compliance autopilot - auto-draft DPIA/ROPA/retention per market ([#1199](https://github.com/ArchiveHeritageGroup/heratio/issues/1199))
- Storytelling engine - trails, "on this day", school packs ([#1202](https://github.com/ArchiveHeritageGroup/heratio/issues/1202))
- Accessibility-first tours - described, contrast, signing ([#1194](https://github.com/ArchiveHeritageGroup/heratio/issues/1194))

## Phase 3 - Connected (mid-long term)

One graph, many institutions; the physical and virtual bridged.

- Unified G/L/A/M knowledge graph - RiC + CIDOC-CRM + KM ([#1197](https://github.com/ArchiveHeritageGroup/heratio/issues/1197))
- Federated GLAM network - opt-in union catalogue + inter-institution loans ([#1203](https://github.com/ArchiveHeritageGroup/heratio/issues/1203), [#1155](https://github.com/ArchiveHeritageGroup/heratio/issues/1155))
- Cross-institution loan / touring-exhibition planning ([#1190](https://github.com/ArchiveHeritageGroup/heratio/issues/1190))
- RiC: exhibition as a first-class Activity, mapped via relations ([#1195](https://github.com/ArchiveHeritageGroup/heratio/issues/1195))
- On-site AR companion ([#1191](https://github.com/ArchiveHeritageGroup/heratio/issues/1191))
- Live virtual openings - ticketed multi-user events ([#1192](https://github.com/ArchiveHeritageGroup/heratio/issues/1192))
- Gaussian-splat photoreal capture ([#1193](https://github.com/ArchiveHeritageGroup/heratio/issues/1193))

## Phase 4 - North Stars (long term / mission)

Five pillars: **Open, Preserved, Just, Alive, True.**

- Open: world heritage graph / open memory protocol ([#1204](https://github.com/ArchiveHeritageGroup/heratio/issues/1204)); every museum for everyone ([#1211](https://github.com/ArchiveHeritageGroup/heratio/issues/1211))
- Preserved: race against loss ([#1205](https://github.com/ArchiveHeritageGroup/heratio/issues/1205)); walk through what no longer exists ([#1206](https://github.com/ArchiveHeritageGroup/heratio/issues/1206))
- Just: repatriation + virtual return ([#1207](https://github.com/ArchiveHeritageGroup/heratio/issues/1207))
- Alive: a culture you can talk to + language revival ([#1208](https://github.com/ArchiveHeritageGroup/heratio/issues/1208)); generative scholarship ([#1210](https://github.com/ArchiveHeritageGroup/heratio/issues/1210))
- True: the truth anchor - verifiable authenticity ([#1209](https://github.com/ArchiveHeritageGroup/heratio/issues/1209))

## Apex

**That humanity never forgets** ([#1212](https://github.com/ArchiveHeritageGroup/heratio/issues/1212)).
One living, trusted memory of all human culture - captured before it is lost, woven into a single graph, provable as true, and open to everyone on Earth, forever. A memory that speaks.

## Sequencing logic

1. **Phase 1** is fast and compounding - it makes existing data smarter and is the clearest near-term value.
2. **Phase 2** earns institutional trust (operations + provenance + compliance) - the table stakes for being the system of record.
3. **Phase 3** unlocks network effects - the unified graph + federation are the platform's moat.
4. **Phase 4 / Apex** are the mission the network exists to serve, pursued *with* communities, not for them.
