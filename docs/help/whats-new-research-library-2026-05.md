# What's New — Research Portal & Library

A summary of recently shipped features, with links to each guide.

## Research portal

- **Journal Builder** — run an institutional scholarly journal (issues → articles → table of contents → publish) *or* draft a manuscript toward an external target journal with submission checks. See *Journal Builder*.
- **Lecture Builder** — author lectures in three modes: curriculum content (feeds Training), public talk/seminar records (speaker, schedule, recording), and standalone authored lectures with sections + media + resources. See *Lecture Builder*.
- **Where to Publish (Target-Journal Directory)** — a directory of journals to publish *to*, each with subject scope and submission rules, seeded with DHET-accredited journals (South-African accreditation module; the directory core is market-neutral). Drives the manuscript builder's validation + best-fit suggestions. See *Where to Publish*.
- **Training & LMS** — a generic, reusable learning-management module: build courses (role/audience, language, pass mark) from curriculum lectures, add a multiple-choice assessment, enrol learners, track progress, and issue certificates on passing. Institution-neutral. See *Training Curriculum & LMS*.

## Library

- **Acquisitions, Vendors, ODI Scorecard, Serials, Overdue** are now linked from the Library hub, each with a user guide. See *Library Acquisitions*, *Library Serials*, *ODI Conformance Scorecard*, *Overdue Items and Notices*, and the *Vendor* guide.

## Digital objects

- **Full embedded image metadata** — the complete EXIF / IPTC / XMP / GPS / MakerNotes / ICC tag set is now captured and shown in an "All metadata" panel (grouped, filterable; GPS hidden for non-administrators), not just a curated subset. See *Embedded Image Metadata (Full Capture)*.

## Fixes

- MARC21 import now decodes the `001` control field and `020$a` ISBN, and preserves repeated fields (e.g. multiple `650` subjects). Inter-library-loan borrow-lane transitions (`pending → requested`) are restored.

> Tip: use the search box at the top of Help to jump straight to any of these guides.
