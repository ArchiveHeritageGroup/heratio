# v1.154.365 (2026-07-17)

feat(portable-export): #1390 #2 - standalone portable-viewer generator. New packages/ahg-portable-export/standalone/heratio-portable-gen.py rebuilds a Heratio-style offline viewer from an OCFL object (reads inventory.json -> latest version -> content) or a flat CSV (AtoM/ISAD-compatible columns + optional assets), with ZERO Heratio present - pure Python 3 stdlib, no pip/framework/server/network. Emits a self-contained index.html with data INLINED (opens by double-click from file://) plus data/records.json, assets/, SHA256SUMS + verify.sh. Closes the reconstructability gap: a collection preserved as OCFL is turnable back into a browsable, searchable archive by anyone, indefinitely. Verified both CSV + OCFL modes end-to-end (tree/search/detail render, verify.sh OK). +README, user-guide cross-reference
