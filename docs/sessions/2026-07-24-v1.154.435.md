# v1.154.435 (2026-07-24)

v1.154.435: feat(#1425): give the RiC record page the ISAD action bar, at the bottom.

The RiC-O record page (show-ric) had a small custom action block at the TOP (Add RiC child + Edit (RiC-O)). Replaced it with the same _actions-bar the ISAD show page uses, moved to the BOTTOM of the page - so RiC and ISAD records now offer identical actions: Edit, Add new + the per-standard child split-button (RiC-O / ISAD / DACS / RAD / MODS / Dublin Core), Duplicate, Move, the More dropdown and Print. Edit routes RiC records back to the RiC-O editor via the existing standard-edit redirect. The RiC view toggle stays at the top. Verified: the bar renders below the record content (after the JSON-LD panel) with all actions; old top 'Edit (RiC-O)' button removed; show-ric compiles.
