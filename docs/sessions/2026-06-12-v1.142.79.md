# v1.142.79 (2026-06-12)

#1244 web-archive consolidation: merge the two duplicate (route-colliding) web-archive implementations into ONE surface. ahg-scan owns the single /admin/web-archive (both capture modes: submit-a-URL + capture-this-record's-page); ahg-core provides the shared engine (WarcCaptureService with SSRF + subresource capture, WarcReplayService with the tested WARC parser + replay). One warc_capture table (mode + submitted_url columns added idempotently); the ahg-core WebArchiveController + ahg-scan duplicate services/table-installer removed. No data loss (both were empty); collision gone; replay serves only from the WARC. #1244 stays open (off-host assets).
