> Heratio Help Center article. Category: Research / Collaboration.

## Overview
Team workspaces enable collaborative research with shared resources, discussions, and member management.

## Features
- **Create Workspace** with name, description, visibility (private/members/public)
- **Members** with role management (viewer/member/editor/admin/owner)
- **Invite** researchers by email with TomSelect search
- **Discussions** with create/edit/delete and reply counts
- **Shared Resources** with type (collection/link/object), edit and remove
- **Shared Collections** linked via workspace resources
- **About Card** showing creation date and your role

## Member Roles
| Role | View | Comment | Add Resources | Manage Members |
|------|------|---------|---------------|----------------|
| Viewer | Yes | No | No | No |
| Member | Yes | Yes | No | No |
| Editor | Yes | Yes | Yes | No |
| Admin | Yes | Yes | Yes | Yes |
| Owner | Yes | Yes | Yes | Yes |
