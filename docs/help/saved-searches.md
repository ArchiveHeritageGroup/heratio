> Heratio Help Center article. Category: Research / Search.

## Overview
Save search queries for quick re-execution and track changes over time.

## Features
- **Save** from GLAM browse or manually enter queries
- **Run** saved searches with one click
- **Snapshot** current results as a baseline
- **Diff** compare current results against last snapshot (added/removed/unchanged)
- **Citation ID** for referencing searches
- **Alerts** on/off for new result notifications

## Saving Searches
From GLAM browse: click Save Search, enter a name. From saved searches page: enter name and query string manually.
