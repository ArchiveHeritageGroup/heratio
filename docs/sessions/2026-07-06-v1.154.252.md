# v1.154.252 (2026-07-06)

fix(help): make the 'Manage links' button visible on mobile. It was crammed into the permalink input-group (slid off-screen on narrow widths) and the link routes required the stricter 'admin' middleware while the button showed for any logged-in user. Moved it to a standalone, always-visible button that wraps below the permalink on mobile, and aligned the routes to 'auth' (matching HelpArticleService::isAdmin() = logged-in). Shows only when signed in.
