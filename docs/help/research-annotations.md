> Heratio Help Center article. Category: Research / Notes.

## Overview
Create research notes linked to archival items, with tags, visibility controls, and export.

## Features
- **Card Grid** layout with thumbnails
- **Search** and filter by visibility (private/shared/public)
- **Tag Cloud** with clickable filters and counts
- **Link** notes to archival descriptions, actors, repositories
- **Export** selected notes as PDF or CSV
- **Per-note** PDF/CSV export and permalink

## Creating Notes
Click Add Note or create while browsing archival descriptions. Set title, content, visibility, and tags.
