> Heratio Help Center article. Category: Research / Reproductions.

## Overview
Request copies or scans of archival materials with structured workflow.

## Creating a Request
1. Click New Request
2. Select purpose, delivery method, urgency
3. Optionally add first item with TomSelect search
4. Submit for processing

## Managing Items
After creation, add items with type (scan/photocopy/photograph/digital), format (PDF/TIFF/JPEG/PNG), quantity, and special instructions.

## Workflow
Draft → Submitted → Quoted → Approved → Processing → Completed

Cancel requests in draft or submitted status.
