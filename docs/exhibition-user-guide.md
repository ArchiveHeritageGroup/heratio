# Exhibition Management

## User Guide

Plan, organize, and manage exhibitions from concept to closing, including object selection, storyline creation, and event scheduling.

## Exhibition Status Workflow

Exhibitions progress through defined stages:

```
+----------+    +----------+    +-------------+    +--------------+
|  CONCEPT |--->| PLANNING |--->| PREPARATION |--->| INSTALLATION |
+----------+    +----------+    +-------------+    +--------------+
     |               |               |                    |
     v               v               v                    v
  Define          Finalize       Design              Install
  theme           object list    layout              objects
  Preliminary     Submit loans   Write text          Set lighting
  budget          Confirm team   Commission          Complete
                                 mounts              condition reports
                    |
                    v
              +---------+    +---------+    +----------+
              |  OPEN   |--->| CLOSING |--->|  CLOSED  |
              +---------+    +---------+    +----------+
                   |              |               |
                   v              v               v
               Monitor        Final           Deinstall
               visitors       reports         Return loans
               Events         Photography     Archive docs
                                             Budget reconcile
                                                  |
                                                  v
                                            +----------+
                                            | ARCHIVED |
                                            +----------+

Status Colors:
  Concept      = Gray
  Planning     = Blue
  Preparation  = Orange
  Installation = Purple
  Open         = Green
  Closing      = Red-Orange
  Closed       = Brown
  Archived     = Gray-Blue
  Canceled     = Red
```

### Changing Status

1. Open the exhibition detail page
2. Valid status transitions appear as buttons
3. Click the desired transition
4. Optionally add a reason for the change

```
Current Status: [PLANNING]

Available Transitions:
  [Return to Concept]  [Begin Preparation]  [Cancel Exhibition]
```

+-------------------+-----------------------------------------------+

## Common Tasks

### Finding an Exhibition

1. Go to **Exhibitions** --> **Browse**
2. Use filters:
   - Status (Open, Planning, etc.)
   - Type (Temporary, Permanent, etc.)
   - Year
   - Search by title/theme

### Extending an Exhibition

1. Open the exhibition
2. Edit the closing date
3. Update status if needed (e.g., from Closing back to Open)

### Canceling an Exhibition

1. Open the exhibition
2. Click **Cancel Exhibition**
3. Provide a reason for cancellation
4. Objects will be released

## Tips and Best Practices
```
+--------------------------------+----------------------------------+
|          DO                    |           DON'T                  |
+--------------------------------+----------------------------------+
| Plan well in advance           | Rush through planning phases     |
| Keep object list updated       | Forget to track loans            |
| Document everything            | Skip condition reports           |
| Use checklists systematically  | Rely on memory                   |
| Create multiple storylines     | Neglect accessibility options    |
| Track insurance values         | Underestimate budget             |
| Archive after closing          | Delete exhibition data           |
+--------------------------------+----------------------------------+
```
