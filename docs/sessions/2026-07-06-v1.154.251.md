# v1.154.251 (2026-07-06)

feat(help): admin link-manager UI for article cross-links. New admin page /help/article/{slug}/links — searchable dropdown (or paste a /help/article URL) to add a bidirectional link, Add & save, see the Linked-articles list, remove with x, repeat. A 'Links' button (next to Copy link) opens it for admins. help_article_link gains a source column so manual UI links survive the markdown re-ingest; new add/remove/search/resolve service methods. New 'Linking help articles' help doc. Verified on dev: routes register, page admin-gated (403 not 500), add/search/remove/resolve + bidirectional all work.
