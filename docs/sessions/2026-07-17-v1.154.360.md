# v1.154.360 (2026-07-17)

feat(portable-export): #1390 - authenticity-carrying offline packages. Every bundle now ships SHA256SUMS (per-file SHA-256, sha256sum -c compatible) + data/fixity.json (rich per-file manifest with stored-checksum cross-reference), carries any persisted C2PA Content Credentials to data/c2pa/ plus on-disk .c2pa.json sidecars next to their asset, and includes verify.sh (one-command CLI check via sha256sum/shasum) + verify.html (self-contained in-browser SubtleCrypto verifier) so a recipient can confirm OFFLINE - no network, no Heratio - that nothing was tampered with. Verified end-to-end on dev: OK on intact file, FAILED on a tampered byte, C2PA manifest carried. README + help guide updated
