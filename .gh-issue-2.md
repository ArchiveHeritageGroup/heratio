Title: PoC Ed25519 signing of inference manifests

Description:

Implement a proof-of-concept that computes an Ed25519 signature over a canonical inference manifest at InferenceService::record() time and stores the signature and signer_key_id in the `ahg_ai_inference` row and RDF graph.

Proposal:
- Generate canonical manifest JSON for each inference (id, timestamp, input_hash, output_hash, model_manifest, provider, confidence).
- Use libsodium (PHP sodium_* functions) or an Ed25519 PHP library to sign the canonical bytes with an operator-managed private key. Store the key reference (signer_key_id) but not the private key in the DB.
- Extend the `ahg_ai_inference` table to hold `signature` (text) and `signer_key_id` (string/uuid).
- Store the signature also in the RDF graph as a signature triple tied to the inference node.

Migration and tests:
- Migration to add signature + signer_key_id columns.
- Unit tests to validate signature generation and verification using a test key pair.

Files referenced:
- packages/ahg-provenance-ai/src/Services/InferenceService.php
- docs/ai-governance-issue.md

Acceptance criteria:
- InferenceService::record() signs manifests and persists signature + signer_key_id.
- Tests demonstrate verification of signature against public key.
