# v1.154.309 (2026-07-13)

chore(deploy): stop tracking per-instance generated branding CSS - public/css/ahg-theme-dynamic.css and public/vendor/ahg-theme-b5/css/ahg-generated.css are rewritten by SettingsController::regenerateThemeCss() from each instance's theme settings, so every branded instance ended up with local modifications that blocked git pull (the recurring deploy blocker). Gitignore + untrack them; instances keep their generated files (now untracked) and regenerate on the next theme-settings save
