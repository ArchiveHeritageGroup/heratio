# Donor Agreement Management

## A Guide for Archivists

---

## What is This For?

When someone **donates materials** to your archive, you need a formal agreement. This system helps you:
- Keep track of all donors
- Create and store donation agreements
- Remember important follow-ups
- Connect donations to the actual records

**Note:** For vendor contracts, service agreements, and institutional partnerships, see [Contract Management](contract-management-user-guide.md).

## Types of Donor Agreements

| Type | When to Use | Witness Needed? |
|------|-------------|-----------------|
| **Deed of Gift** | Donor gives materials permanently, no strings attached | Yes |
| **Deed of Donation** | Formal donation under South African law | Yes |
| **Deposit** | Materials stay with you but donor keeps ownership | No |
| **Loan** | Temporary - for exhibition or research | No |
| **Purchase** | You are buying the materials | No |
| **Bequest** | Received through someone's will | Yes |
| **Transfer** | Materials transferred from another institution | No |
| **Custody** | Temporary custody pending decision | No |
| **Access Agreement** | Special access arrangements for sensitive materials | No |
| **Reproduction Agreement** | Rights for reproduction of donated materials | No |

## Adding a Logo to Agreements

You can add your organization's logo to any agreement for professional presentation.

### How to Add a Logo

1. Go to the agreement (Add or Edit)
2. In the **Agreement Details** section, find **Agreement Logo**
3. Click **Choose File** and select your logo
4. Supported formats: JPG, PNG, GIF, WebP
5. Save the agreement

### Logo Display

- The logo appears at the top of the agreement when viewing
- Useful for printed/PDF versions
- Recommended size: Max height 120px

### Removing a Logo

1. Edit the agreement
2. Check the **Remove logo** checkbox
3. Save the agreement

## Using Templates

Save time by creating template agreements:

### Creating a Template

1. Create an agreement with all standard terms filled in
2. Check the **Save as Template** checkbox
3. Save

### Using a Template

1. When creating a new agreement, find a suitable template
2. Clone the template
3. Modify details for the specific situation
4. Uncheck **Save as Template** for the new agreement

## Creating a New Donor

1. Go to **Admin > Donor Management**
2. Click **New Donor**
3. Fill in their details:

| Field | What to Enter |
|-------|---------------|
| Name | Full name |
| Organisation | Company or institution (if any) |
| Address | Postal address |
| Phone | Contact number |
| Email | Email address |
| Notes | How you know them, relationship history |

4. Click **Save**

---

## Creating an Agreement

1. Go to the donor's record
2. Click **New Agreement**
3. Fill in the form:

| Field | Example |
|-------|---------|
| Title | Smith Family Papers 2026 |
| Type | Deed of Gift |
| Date | 15 January 2026 |
| Description | Personal papers, photographs, and correspondence of John Smith (1920-2005) |
| Special Terms | No access to medical records for 25 years |
| Status | Draft |

4. Click **Save**

### Generate the Document

1. From the agreement, click **Generate Document**
2. Choose PDF or Word format
3. Review the document
4. Print two copies

### Upload Signed Copy

1. After getting signatures, scan the document
2. Go back to the agreement record
3. Click **Upload Document**
4. Select your scanned PDF

## Linking to Records

After you've processed the donated materials:

```
+------------------+     +------------------+
|    AGREEMENT     |     |    ARCHIVAL      |
|                  |     |    RECORDS       |
|  Smith Family    |---->|                  |
|  Papers 2026     |     |  - Letters       |
|                  |     |  - Photos        |
|                  |     |  - Diaries       |
+------------------+     +------------------+
```

1. Go to the agreement
2. Click **Link Records**
3. Search for the archival descriptions
4. Select the ones that came from this donation
5. Click **Link**

Now you can always see which records came from which donation.

## Setting Reminders

Never forget an important follow-up:

1. Go to any agreement
2. Click **Add Reminder**
3. Enter:
   - What needs to be done
   - Due date
   - Who should be reminded (yourself or colleague)

Reminders appear on the dashboard and can send email notifications.

### Common Reminders

| Reminder | When |
|----------|------|
| Contact donor about additional materials | 6 months after donation |
| Review access restrictions | Before expiry date |
| Send thank you letter | 1 week after receipt |
| Follow up on promised items | 3 months after initial contact |

## Access Restrictions

Some donations come with conditions. Record these carefully:

### Types of Restrictions

| Restriction | What It Means |
|-------------|---------------|
| **Embargo** | No one can see it until a certain date |
| **Permission Required** | Must ask donor before allowing access |
| **Reading Room Only** | Can view but not copy |
| **Anonymised** | Names must be hidden |

### How to Add Restrictions

1. Go to the agreement
2. Click **Add Restriction**
3. Select the type
4. Enter details (dates, conditions)
5. Click **Save**

These restrictions automatically apply to all linked records.

## Quick Reference

### Agreement Statuses

| Status | Meaning |
|--------|---------|
| Draft | Still being prepared |
| Pending | Waiting for signatures |
| Active | Signed and in effect |
| Closed | Completed or superseded |

### Best Practices

**Always get it in writing**
- Even for small donations
- Protects both you and the donor

**Upload signed copies immediately**
- Don't let paper pile up
- Digital copies are searchable

**Link to records as you process**
- Easier to do as you go
- Harder to remember later

**Set reminders proactively**
- Thank you letters
- Follow-up conversations
- Restriction expiry dates

## Donor vs Contract Management

| Use Donor Agreements for: | Use Contract Management for: |
|---------------------------|------------------------------|
| Donations from individuals | Vendor service agreements |
| Gifts and bequests | Collaboration partnerships |
| Deposits and loans TO you | SLAs with service providers |
| Access restrictions on donated materials | NDAs and data processing agreements |
| Reproduction rights from donors | Maintenance and lease contracts |
