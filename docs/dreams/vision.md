# The Apex Vision

> **That humanity never forgets.**
> One living, trusted memory of all human culture - every people, every place, every story - captured before it is lost, woven into a single graph, provable as true, and open to everyone on Earth, forever. Not a vault of dead records. A memory that speaks: you can walk it, question it, and hear any culture in its own voice.

Tracker: [#1212](https://github.com/ArchiveHeritageGroup/heratio/issues/1212). North: **preserve it, link it, prove it, open it.**

## The five pillars (north-stars)

Everything Heratio builds ladders up to five qualities the memory layer of humanity must hold at once.

### Open
A shared memory no one owns - federated, accountable, reachable by anyone.
- World heritage graph / open memory protocol - the "TCP/IP of cultural memory" ([#1204](https://github.com/ArchiveHeritageGroup/heratio/issues/1204))
- Every museum for everyone - universal, multilingual, any-device, free access ([#1211](https://github.com/ArchiveHeritageGroup/heratio/issues/1211))

### Preserved
Capture what is at risk before it is gone; bring back what is already lost.
- Race against loss - global endangered-heritage capture network ([#1205](https://github.com/ArchiveHeritageGroup/heratio/issues/1205))
- Walk through what no longer exists - reconstruct lost places from evidence ([#1206](https://github.com/ArchiveHeritageGroup/heratio/issues/1206))

### Just
Memory as repair, built with communities, not for them.
- Repatriation engine - trace displaced heritage home + virtual return ([#1207](https://github.com/ArchiveHeritageGroup/heratio/issues/1207))

### Alive
Heritage you can interact with, not just store.
- A culture you can talk to + endangered-language revival ([#1208](https://github.com/ArchiveHeritageGroup/heratio/issues/1208))
- Generative scholarship - AI finds connections no human spotted ([#1210](https://github.com/ArchiveHeritageGroup/heratio/issues/1210))

### True
The trusted record in an age of fakes.
- The truth anchor - verifiable authenticity for primary sources ([#1209](https://github.com/ArchiveHeritageGroup/heratio/issues/1209))

## The questions we cannot get wrong

- **Governance** - who governs a shared memory of humanity? It must be federated and accountable; concentration of memory is concentration of power.
- **Honest AI** - let the machine help without letting it lie: every machine contribution labelled, sourced, reversible (provenance on the tools, not only the artefacts).
- **Whose story** - description is never neutral; make room for contested histories and multiple voices, not one official record.
- **Forever economics** - preservation is a permanent obligation on impermanent budgets; we have not solved this at scale.
