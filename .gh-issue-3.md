Title: AI Inventory UI: governance dashboard stub

Description:

Create a basic AI Inventory & Governance dashboard page that lists LLM configurations and recent inference stats. This is a UI stub to enable operator visibility per the AI Governance plan.

Proposal:
- Backend endpoints:
  - GET /admin/governance/models -> returns list of ahg_llm_config rows with last_used timestamp and model_manifest summary.
  - GET /admin/governance/inferences -> returns recent inferences with id, timestamp, confidence, signed boolean.
- Frontend:
  - Blade view `resources/views/admin/governance/index.blade.php` that renders the lists and links to inference detail.
  - Route and controller `Admin\GovernanceController@index` to serve the page.
- Tests: route and controller unit tests to ensure data is returned.

Files referenced:
- docs/ai-governance-issue.md
- packages/ahg-ai-services/src/Services/LlmService.php
- packages/ahg-provenance-ai/src/Services/InferenceService.php

Acceptance criteria:
- Admin page accessible to admin users and shows model list + recent inference stats.
- API endpoints return JSON for use by future JS enhancements.
