# v1.154.382 (2026-07-21)

fix(nav): show the Articles nav icon on heratio.org. The Articles top-nav link (and its admin-menu entry) is gated to 'marketing' homepage mode, whose host fallback only recognised heratio.theahg.co.za - but the canonical public domain is now heratio.org (theahg.co.za redirects to it), so heratio.org fell through to 'institutional' and the Articles icon was hidden. Add heratio.org + www.heratio.org to the marketing-host list in header.blade.php and ahg-admin-menu.blade.php; the heratio.homepage_mode config override still wins when set
